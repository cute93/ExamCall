📋 ExamCall

같은 WiFi 환경의 태블릿 간 이름 호출 앱

Android 7.0+ WiFi 필요 한국어 TTS 서버 여러 대 지원

📌 앱 개요

🖥️ 서버 태블릿

  • 호출할 이름 목록 추가 / 삭제
  • 클라이언트가 호출하면 이름을 크게 표시
  • 한국어 TTS로 "OOO 선생님 호출입니다" 읽어줌
  • 여러 대 동시 운용 가능

📱 클라이언트 태블릿

  • WiFi 내 서버를 자동으로 탐색
  • 연결할 서버 선택
  • 이름 버튼을 눌러 서버에 호출 전송
  • 언제든 다른 서버로 전환 가능

📥 설치 방법

1. GitHub Releases에서 app-release.apk 다운로드

2. 태블릿에서 설정 → 보안 → 출처를 알 수 없는 앱 허용 후 APK 설치

3. 한국어 TTS가 없다면 설치 필요:

설정 → 일반 관리 → 텍스트 음성 변환 → Google TTS → 언어 → 한국어 다운로드

🖥️ 서버 태블릿 사용법

1

앱 실행 → "서버 (명단 관리)" 선택

역할 선택 화면에서 서버를 탭합니다.

2

서버 이름 입력

클라이언트 목록에 표시될 서버 이름을 입력합니다. (예: 1반, 시험장A)

💡 같은 WiFi에 서버 여러 대 운용 시 이름으로 구별됩니다
3

명단 추가

하단 입력창에 이름을 입력하고 추가 버튼을 탭합니다. 이름은 자동으로 클라이언트에 동기화됩니다.

4

호출 수신

클라이언트가 버튼을 누르면 이름이 화면에 크게 표시되고 "OOO 선생님 호출입니다" 음성이 재생됩니다. 10초 후 자동으로 사라집니다.

📱 클라이언트 태블릿 사용법

1

앱 실행 → "클라이언트 (호출 버튼)" 선택

역할 선택 화면에서 클라이언트를 탭합니다.

2

서버 선택

같은 WiFi 내 서버가 자동으로 탐색되어 목록에 표시됩니다. 원하는 서버를 탭합니다.

💡 서버가 보이지 않으면 서버 태블릿이 켜져 있는지 확인하세요
3

이름 버튼 탭 → 호출

서버에 등록된 이름이 버튼으로 표시됩니다. 버튼을 탭하면 즉시 서버에 호출이 전송됩니다.

4

서버 변경

상단 "서버 변경" 버튼을 탭하면 서버 목록으로 돌아가 다른 서버에 연결할 수 있습니다.

❓ 자주 묻는 질문

Q. 서버가 클라이언트 목록에 나타나지 않아요.
서버 태블릿과 클라이언트 태블릿이 같은 WiFi에 연결되어 있는지 확인하세요. 공유기의 AP 격리(AP Isolation) 기능이 켜져 있으면 통신이 차단됩니다.
Q. TTS 음성이 영어로 나와요.
설정 → 일반 관리 → 텍스트 음성 변환에서 Google TTS 엔진의 한국어 데이터를 다운로드하세요.
Q. 앱을 재시작하면 명단이 사라지나요?
아니요. 명단은 기기 내부에 자동 저장되어 재시작 후에도 유지됩니다.
Q. 서버 여러 대를 동시에 운용할 수 있나요?
가능합니다. 각 서버에 다른 이름을 부여하면 클라이언트에서 목록으로 표시되어 원하는 서버를 선택할 수 있습니다.
Q. 호출 알림이 10초보다 길게 표시되게 할 수 있나요?
현재 버전은 10초 자동 소멸이 고정되어 있습니다. "확인" 버튼을 탭하면 즉시 닫을 수 있습니다.
